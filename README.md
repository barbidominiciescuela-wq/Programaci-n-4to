# 📖 Descripción

Este repositorio reúne todos los contenidos, ejercicios, prácticas y proyectos desarrollados durante la materia **Programación** de **4.º año**.

Durante la primera parte del curso aprenderemos los fundamentos de **Python**, mientras que en la segunda etapa aplicaremos esos conocimientos al **desarrollo de videojuegos**.
